# Offside Football Analytics Datathon

## Objective

Predict whether a football player will score in a football match.

## Dataset

* Train: 1.3M+ rows
* Test: 565K+ rows

## Models Used

* LightGBM
* CatBoost

## Feature Engineering

* attacking_index
* creative_index
* market value features
* match context features

## Results

* Best Validation AP: 0.42779
* Public Leaderboard Score: 0.41750
